# skycheck

See what's visible in the night sky tonight based on your location.

## What it does
Enter your location and get:
- Visible planets and their positions
- Current moon phase
- Sunrise/sunset times
- (more features coming)

## Built with
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- astronomy-engine

## Run locally
```bash
git clone https://github.com/kirenia/skycheck.git
cd skycheck
npm install
npm run dev
```

Open http://localhost:3000
